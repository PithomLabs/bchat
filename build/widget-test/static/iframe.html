iframe Wrapper Test

This page demonstrates the iframe embedding method. The widget is loaded inside an iframe, isolating it from the parent page.

The chat widget is embedded in the iframe on the right. The chat panel opens automatically in iframe mode.

The widget was embedded using this code:

<iframe
  src="https://bchat.fly.dev/widget/rizal/iframe?color=%230d9488&welcome=Hi!%20How%20can%20I%20help%20you%20today%3F&companyName=rizal"
  style="width: 100%; height: 500px; border: none;"
  title="Chat Widget"
></iframe>

How it works:

  1. The iframe loads /widget/rizal/iframe from bchat
  2. Query parameters customize color, welcome message, etc.
  3. The server returns a complete HTML page with the widget
  4. The widget auto-opens in iframe mode (no floating button)
  5. Full DOM isolation - no CSS/JS conflicts with parent page

When to use iframe:

  • CMS platforms that block external scripts
  • Sites with strict Content Security Policy (CSP)
  • When you need complete style isolation
  • Embedding in sandboxed environments
