Simple Embed Test

This page demonstrates the JavaScript widget injection method. The widget loads and injects a floating chat button into the page.

Look at the bottom-right corner - you should see a teal chat button. Click it to open the chat panel.

The widget was embedded using this code:

<script src="https://bchat.fly.dev/widget/rizal/embed.js"></script>
<script>
  AgentChatWidget.init({
    tenant: "rizal",
    baseUrl: "https://bchat.fly.dev",
    color: "#0d9488",
    position: "bottom-right",
    welcomeMessage: "Hi! How can I help you today?",
    companyName: "rizal"
  });
</script>
↓

How it works:

  1. The script tag loads embed.js from the bchat server
  2. AgentChatWidget.init() merges your config with defaults
  3. The widget injects CSS and DOM elements into your page
  4. A floating button appears; clicking it opens the chat panel
  5. Messages are sent to /api/v1/agent/rizal/chat/ext
