Simple Embed

Tests the JavaScript widget injection method. The widget script is loaded and AgentChatWidget.init() is called. A floating chat button appears in the bottom-right corner.

Best for: Most websites - the widget injects directly into the page DOM with full customization options.

iframe Wrapper

Tests the iframe embedding method. The widget is loaded inside an <iframe> element, isolating it from the parent page.

Best for: Platforms with strict CSP policies, CMS systems that block external scripts, or when DOM isolation is required.
