Embed Code (Same as Agent Admin)

This site uses the same embed code format generated by Agent Admin:

<script src="https://your-domain.com/widget/scraper/embed.js"></script> <script> AgentChatWidget.init({ tenant: "scraper", baseUrl: "https://your-domain.com", color: "#0d9488", position: "bottom-right", welcomeMessage: "Hi! How can I help you today?", companyName: "Pithom Labs" }); </script>

Test Instructions

  • Build widget: cd widget && npm run build
  • Copy to site: cp dist/embed.min.js site/
  • Start server: task run:rag or task run:rag:mock
  • Serve this site: cd widget/site && python -m http.server 8080
  • Open http://localhost:8080 in browser
  • Click the chat button in the bottom-right corner
